# コーディングスタイルスキル

プロジェクトのコーディング規約を適用します。

## 参照ドキュメント

以下のドキュメントを参照してコーディング規約を適用してください：

- @docs/reference/technical/coding-style.md - コーディングスタイル全般

## 適用ルール

1. 命名規則を遵守（camelCase/PascalCase/UPPER_SNAKE）
2. フォーマットルールを適用
3. TypeScript の型定義ルールに従う
4. 関数・コメントのガイドラインを遵守
