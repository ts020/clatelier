# 用語集（ユビキタス言語）

コード・ドキュメント・会話で統一して使用する用語です。

## コア用語

| 日本語 | 英語（コード） | 定義 |
|--------|---------------|------|
| スキル | Skill | Claudeの能力を拡張する知識・ガイドライン定義 |
| エージェント | Agent | 専門タスクを実行する独立したAIアシスタント |
| コマンド | Command | ユーザーが明示的に実行するワークフロー |
| スキルグラフ | SkillGraph | スキルの視覚的・構造的表現 |
| ノード | SkillNode | スキルグラフを構成する要素 |
| エッジ | SkillEdge | ノード間の接続・関係 |

## ノードタイプ

| タイプ | 説明 | 用途 |
|--------|------|------|
| instruction | 手順・指示 | Claudeが従うべきステップ |
| tool | ツール定義 | 使用を許可するツール |
| example | 使用例 | 具体的な例示 |
| decision | 分岐条件 | 条件による処理分岐 |

## プロセス用語

| 日本語 | 英語 | 定義 |
|--------|------|------|
| レビューループ | Review Loop | 指摘がゼロになるまで繰り返すレビュープロセス |
| フェーズ | Phase | 開発プロセスの段階（設計/実装/品質確認/完了） |
| 指摘 | Issue | レビューで発見された問題点 |

## 重要度レベル

| レベル | 説明 | 対応 |
|--------|------|------|
| CRITICAL | セキュリティ・バグ | 必須修正 |
| WARNING | 規約違反・品質問題 | 修正推奨 |
| INFO | 改善提案 | 任意対応 |
